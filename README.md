
# Django CRUD API Project with postgre SQL .db 
 Project Status
 Core CRUD APIs implemented with JSON
 Currently improving backend features (PATCH, PUT, validation)
 Frontend integration using fetch and DOM in progress

Features
 Create a new contact (POST)
 Get all contacts (GET)
 Update a contact (PUT / PATCH)
 Delete a contact (DELETE)
 Input validation for missing fields
 JSON responses for all API endpoints
 Basic frontend integration using HTML and JavaScript fetch
